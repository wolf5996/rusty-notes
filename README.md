# Learning Rust: The Book

A structured follow-through of [The Rust Programming Language](https://rust-book.cs.brown.edu/)—notes, exercises, and code from each chapter.

## Why Rust?

Rust combines high-level ergonomics with low-level control, letting you manage memory without the usual hassle. It's been gaining significant adoption in bioinformatics and systems programming, and offers something rare: an introduction to low-level programming that's actually approachable.
